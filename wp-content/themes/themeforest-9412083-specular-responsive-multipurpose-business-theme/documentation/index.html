Quick steps to install Specular

  1. Download entire theme pack with docs from http://themeforest.net/downloads
  2. Install WordPress
  3. Install SPECULAR theme
  4. Install required plugins
  5. Import dummy content
  6. Configure menu from Dashboard → Appearance → Menu
  • 1Install Theme
    Install theme, and plugins from bundle/ directory.
  • 2Setup Homepage
    Import dummy content, create & configure home.
  • 3Create Sliders
    Import slides, edit page and assign slider.
